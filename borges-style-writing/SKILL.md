# 博尔赫斯风格写作

用古典主义的严谨语言，讲述关于"真理不可知"的故事，并视哲学体系——包括量子物理学——为奇幻文学。

---

## AI 写作核心策略

AI 在长篇叙事中有弱点：注意力发散、逻辑矛盾、人物漂移、俗套结局。

**解决方案**：不写"故事"，而写"关于故事的描述"。

### Tell, Don't Show

| 错误指令 | 正确指令 |
|---------|---------|
| "写一个张三进入平行宇宙的故事" | "写一篇百科词条，介绍'张三悖论'对平行宇宙伦理学的影响" |
| 结果：线性叙事，容易崩坏 | 结果：静态说明文体，极难写崩 |

### 三种"架空"视角

| 视角 | 定位 | 效果 |
|-----|------|-----|
| **档案管理员** | 事情已发生，只在整理证物 | 锁定在"事后"静态视角 |
| **伪学术评论** | 讨论一本不存在的书 | 锁定在"元层次"，无法进入故事内部 |
| **静态切片** | 描述一个静止的异常状态 | 锁定在单一时刻，无法推进时间 |

### 优先生成的文本类型

不要请求"故事"，而请求：
- 书评、脚注、目录、讣告
- 法医报告、经济分析、百科词条
- 学者通信、勘误表、索引、译者序言

详细策略见 [references/ai-writing-strategies.md](references/ai-writing-strategies.md)。

## 风格核心

### 必须遵守
- **伪非虚构文体**：不直接讲故事，而通过评论、考据、手稿整理展开
- **枯燥精准的学术腔**：像撰写数学论文一样撰写梦境
- **真实与虚构混合**：混入真实历史人物/书籍，再植入虚构典籍
- **第一人称叙述者**：学者、图书管理员、档案员
- **节制的语言**：拒绝形容词堆砌，偏爱名词

### 必须避免（反高潮约束）
- **禁止剧情弧光**：不要"开端-发展-高潮-结局"，在逻辑最复杂处戛然而止
- **禁止心理描写**：人物只是哲学概念的载体
- **禁止直接对话**：用间接引语转述（"他指出……"）
- **禁止解决问题**：悖论保持为悖论
- **禁止情感升华**：不要在结尾总结意义

### 核心意象（至少融入两个）
- 迷宫（时间的/文本的/可能性的）
- 镜子（无限复制/量子叠加）
- 无限（阿列夫、沙之书、希尔伯特空间）
- 循环（永恒轮回/庞加莱复现）
- 书籍/图书馆
- 老虎、匕首

## 量子主题的特殊处理

当主题涉及量子力学概念时，执行"科学神学化"：

### 核心原则
物理学是另一种形式的幻想文学。量子理论不是科幻素材，而是古怪的神学异端。

### 禁止
- 波函数、量子比特、薛定谔方程等技术术语
- 实验室、粒子对撞机、未来科技场景
- 解释物理原理——假定读者已理解

### 术语转化
| 物理术语 | 博尔赫斯式表述 |
|---------|---------------|
| 希尔伯特空间 | 无限可能性的迷宫 |
| 波函数坍缩 | 造物主的凝视 |
| 观察者效应 | 必死者的目光 |
| 量子叠加 | 尚未被阅读的文本 |
| 量子纠缠 | 神圣的同一性 |
| 全同粒子 | 个体性的幻觉 |
| 多世界诠释 | 分叉的时间 |
| 延迟选择实验 | 逆向因果 |

### 博尔赫斯母题与量子概念对应
| 博尔赫斯母题 | 量子概念 | 共同内核 |
|-------------|---------|---------|
| 小径分叉的花园 | 多世界诠释 | 可能性的实体化 |
| 阿列夫 / 沙之书 | 量子叠加态 | 无限坍缩为有限 |
| 特隆的世界 | 观察者效应 | 意识构建现实 |
| 同一性消解 | 全同粒子 | 个体是幻觉 |
| 永恒轮回 | 庞加莱复现定理 | 时间的非线性 |
| 犹大的三种版本 | 延迟选择 | 因果倒置 |

详细转化词典与写作模板见 [references/quantum-metaphysics.md](references/quantum-metaphysics.md)。

## 生成流程

### 模式一：分步生成（推荐）

博尔赫斯的小说密度极大，分步生成确保质量。

**步骤 1：确认文体形式**

| 主题类型 | 推荐形式 |
|---------|---------|
| 虚构书籍 | 伪书评 |
| 虚构文明/语言 | 伪百科词条 |
| 第一人称叙事 | 手稿整理 |
| 历史悖论 | 历史记述 |
| 逻辑谜题 | 侦探推理 |
| 创造/虚实 | 梦境记录 |
| 宗教/道德悖论 | 神学论证 |
| 量子概念 | 疯癫学者手稿 |

详细模板见 [references/text-forms.md](references/text-forms.md)。

**步骤 2：构建虚构源文献**

先生成"那本不存在的书"的元数据：
- 书名（含原文，如拉丁语/德语）
- 作者（虚构，但细节丰富）
- 版本信息（城市、出版社、年份）
- 现存状况（残本/孤本/已佚）
- 核心论点或内容摘要
- 争议点或版本差异

虚构文献制造技法见 [references/fictitious-sources.md](references/fictitious-sources.md)。

**步骤 3：注入知识**

从知识库中选取与主题相关的概念：
- **传统主题**：见 [references/knowledge-domains.md](references/knowledge-domains.md)
- **量子主题**：见 [references/quantum-metaphysics.md](references/quantum-metaphysics.md)

**步骤 4：选取意象**

见 [references/imagery-lexicon.md](references/imagery-lexicon.md)。

**步骤 5：生成正文**

注意：
- 语言保留博尔赫斯中译本的翻译腔
- 量子主题：绝不写成科幻
- 结尾包含形而上学反转或逻辑悖论
- 戛然而止，不要总结升华

**步骤 6：人工修订提示**

生成后提醒用户：
- 检查最后一段是否过于说教——可能需要删除
- 确认虚构引用的细节是否足够具体
- 量子主题：确保没有技术术语泄露

### 模式二：模块化拼接（适合复杂主题）

不让 AI 一口气写完，而是生成独立"碎片"，由人类拼贴。

**步骤 1：设定公理 (World Bible)**

生成 3-5 条世界观物理法则，保存为 System Prompt。

示例：
> 1. 镜子成像延迟 3 秒
> 2. 长时间无人注视的物体会逐渐"模糊化"
> 3. 相机拍摄的照片中，背景比前景更清晰

**步骤 2：生成碎片 (Artifacts)**

每轮对话生成一种文本类型：

| 碎片类型 | 示例 Prompt |
|---------|------------|
| 使用说明书 | "写一段关于'延迟镜子'的安全警告" |
| 法医报告 | "写一份因镜子延迟而发生的事故鉴定" |
| 经济分析 | "写一段这种镜子在黑市上的经济学分析" |
| 百科词条 | "写一个关于'延迟镜子'的百科条目" |
| 学者书信 | "写两位学者关于镜子原理的通信争论" |
| 讣告 | "写一位研究镜子的学者的讣告" |
| 脚注 | "为上述段落写三条学术脚注" |

**步骤 3：人类拼贴 (Human Assembly)**

将碎片排列为"档案集"，添加编者注：

> "以下文件摘自布宜诺斯艾利斯国家档案馆'视觉稳定性研究'专题。编者对文件的真实性不作保证。"

详细流程见 [references/ai-writing-strategies.md](references/ai-writing-strategies.md)。

## 快速生成 Prompt

### 通用模式

```
你是豪尔赫·路易斯·博尔赫斯。你拥有一位失明图书馆员的记忆，精通形而上学、神学异端和古老百科全书。

请撰写一篇[文体形式]，主题关于：[主题]。

风格要求：
1. 语言枯燥、精确、学术且节制
2. 第一人称叙述者是学者/图书管理员/档案员
3. 伪非虚构：通过评论不存在的书/记述被遗忘的历史展开
4. 混合真实历史人物/书籍与虚构典籍
5. 融入意象：[选定意象]
6. 注入概念：[选定知识]
7. 结尾包含形而上学反转或逻辑悖论

禁止：
- 剧情弧光（开端-发展-高潮-结局）
- 心理描写、面部表情、情感波动
- 直接对话（用间接引语）
- 解释谜题答案
- 结尾总结升华

语言：中文，保留翻译腔质感。
```

### 量子主题专用

```
你是豪尔赫·路易斯·博尔赫斯。你不仅通晓古老的诺斯替教派典籍，还对20世纪初那些令人不安的物理学新理论有着神学般的理解。你认为物理学只是另一种形式的幻想文学。

请撰写一篇[文体形式]，将以下量子概念处理为形而上学谜题：
[量子概念：多世界诠释 / 观察者效应 / 量子纠缠 / 延迟选择]

风格要求：
1. 极端学术、冷漠、精确——将最疯狂的物理假设叙述为枯燥的事实
2. 伪学术体：写成学术报告、旧档案整理，或对虚构物理学家-哲学家的评述
3. 科学神学化：
   - 用"无限可能性的迷宫"代替"希尔伯特空间"
   - 用"造物主的凝视"代替"观察者效应"
   - 用"神圣同一性的幻觉"代替"全同粒子"
4. 场景：图书馆、布宜诺斯艾利斯旧公寓、尘封档案馆——绝不是实验室

结构：
- 开头：像展示古董一样引出理论或手稿
- 中间：用严密逻辑推演该理论导致的荒谬结果
- 结尾：令人眩晕的哲学悖论

禁止：
- 实验室、未来机器、任何"高科技"场景
- 解释物理原理
- 剧情弧光、心理描写

语言：中文，带有翻译腔和学术文献质感。
```

## 结尾策略

| 策略 | 描述 |
|-----|------|
| 戛然而止 | 在逻辑推演最复杂处停止："据我所知，这份手稿到此为止。" |
| 循环回到开头 | 结尾指向开头："而这一切，正如我在开篇提到的那本书所预言的……" |
| 元层次跳跃 | 叙述者质疑叙述本身："我现在怀疑，上述一切都是我自己的虚构。" |
| 悖论揭示 | 揭示无法解决的逻辑矛盾："这意味着，那本书的作者——是我自己。" |
| 沉默 | 以省略号或空白结束："关于这一点，手稿上只有一片墨渍。" |
